## Install
git clone git@github.com:cdjdesenvolvimento/_Senna.git
cd _senna
php composer.phar self-update
php composer.phar install
cd vendor
git clone https://github.com/Infanatica/InfanaticaCepModule.git
cd..
cd public
cls
php -S 127.0.0.1:8080
cd
